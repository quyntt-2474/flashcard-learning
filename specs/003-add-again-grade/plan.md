## Design Decisions

### 1. Keyboard Shortcut Mapping

Adopt Anki-standard numeric mapping so spaced-repetition users need no re-learning:

| Key | Grade | SM-2 numeric |
|-----|-------|-------------|
| `1` | Again | 0 (reset)   |
| `2` | Hard  | 3           |
| `3` | Good  | 4           |
| `4` | Easy  | 5           |

Existing `Enter` / `ArrowRight` → Good shortcut is **preserved** (spec FR-007).
Numeric shortcuts fire only when `revealed === true` and no input element is focused (reuse existing guard logic).

### 2. GradeBar Layout

- Add `AGAIN` as first entry: `{ grade: 'AGAIN', label: 'Again', color: 'bg-rose-500 hover:bg-rose-600', key: '1' }`
- Shift existing keys: Hard `[2]`, Good `[3]`, Easy `[4]`
- Change grid from `grid-cols-3` → `grid-cols-4`
- At 320 px, four buttons are each ≈ 72 px wide (320 − 2×16 padding − 3×8 gap) / 4 ≈ 68 px — above the 44 px minimum

### 3. SessionSummary Layout

- Add "Again" as the first column (rose colour, matching the button)
- Switch from `grid-cols-3` → `grid-cols-4`
- Data source: `summary.again` (already present in `SessionSummaryData`)

### 4. No Backend Changes

The backend:
- Already accepts `AGAIN` via `POST /sessions/:id/reviews` (Prisma enum + DTO)
- Already counts Again in `complete()` and returns `again: number`
- SM-2 already resets `interval=1`, `repetitions=0` for Again grade
- `accuracyPercent` already excludes Again from correct count

### Step 2 — Keyboard Shortcuts (`frontend/app/study/[sessionId]/page.tsx`)

Add numeric key handlers inside the existing `useEffect`. The guard for input focus
(`tag === "INPUT" || tag === "TEXTAREA"`) and the `revealed` check already exist:

```tsx
useEffect(() => {
  const handler = (e: KeyboardEvent) => {
    const tag = (e.target as HTMLElement).tagName;
    if (tag === "INPUT" || tag === "TEXTAREA") return;

    if (!revealed && e.key === " ") {
      e.preventDefault();
      handleReveal();
    } else if (revealed) {
      if (e.key === "Enter" || e.key === "ArrowRight") {
        e.preventDefault();
        handleGrade("GOOD");
      } else if (e.key === "1") { handleGrade("AGAIN"); }
      else if (e.key === "2")   { handleGrade("HARD");  }
      else if (e.key === "3")   { handleGrade("GOOD");  }
      else if (e.key === "4")   { handleGrade("EASY");  }
    }
  };
  window.addEventListener("keydown", handler);
  return () => window.removeEventListener("keydown", handler);
}, [revealed, handleReveal, handleGrade]);
```
